Antwoorden op Vragen

1. Waar wordt de event listener gekoppeld aan de slider? De event listener wordt in de JavaScript-code aan het eerste slider-element gekoppeld met sliders[0].addEventListener("change", update) en sliders[0].addEventListener("input", update).

2. Waarom moeten we die op twee soorten events koppelen? Zodat we zowel continue updates tijdens het slepen (input) als een definitieve update bij loslaten (change) krijgen.

3. In de CSS file wordt nergens een rode kleur opgegeven, waar wordt dan wel de rode kleur van het blokje ingesteld? De rode kleur wordt programmatorisch ingesteld via JavaScript met colorDemos[0].style.backgroundColor = "red".

4. Waarom schrijven we telkens sliders[0] en colorDemos[0] en niet gewoon sliders of colorDemos? Omdat getElementsByClassName een collectie teruggeeft en we het eerste (en enige) element uit die collectie willen gebruiken.

5. Waar vind je de CSS-properties die programmatorisch werden ingesteld? Deze worden als inline styles weergegeven onder het gedeelte element.style in de Developer Tools.
